# colormixer
An educational demo about color mixing of light.

## Overview
This project uses a clear common anode RGB LED and a diffused through-hole WS2812B LED to demonstrate the science of color mixing. The clear LED allows you to see the individual red, green, and blue LEDs, and the diffused LED mixes the colors a bit better.
